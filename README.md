# 古诗跑酷 · 急速变色龙（水墨版）

一款纯离线、可在手机浏览器直接打开的水墨风跑酷小游戏：操控会"变色"的变色龙在山水长卷间奔跑，通过黑白变形躲避障碍、拾取真实古诗词残句，在"跑得爽"的过程中记住一句句经典诗词。

## 玩法

- **变色躲避**：点击切换白（飞）/黑（奔）两种形态，匹配形态拾取圆形「飞诗」与方形「奔诗」。
- **集诗觉醒**：收满一类诗即可觉醒大招——「飞天」御风高飞、「疾跑」虚空铺路（自带短暂无敌 + 冷却）。
- **对诗模式**：长按调出选字轮盘补全诗句，「墨时」慢放但消耗墨心；难度分教学/入门/进阶/大师。
- **成长系统**：通关集诗得星，商店兑换斗笠/发冠/披风，配合成就与挑战榜复玩。

## 运行

纯离线、零依赖、零联网。直接用浏览器打开 `index.html` 即可游玩（建议手机竖屏）。

## 文件

- `index.html` — 游戏主体（逻辑/界面/音效全部内联）
- `js/landscapes.js` — 程序化国风山水远景引擎（`window.LandscapeArt`）
- `原调音乐-清静古风（轻音乐）.ogg` — 背景音乐
